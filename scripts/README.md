# Development Scripts

- `scripts/bootstrap` - Prepare local development environment.
- `scripts/check` - Check whether code linting passes.
- `scripts/clean` - Delete any build artifacts.
- `scripts/coverage` - Check test coverage.
- `scripts/install` - Install dependencies in a virtual environment.
- `scripts/lint` - Run code linting.
- `scripts/setup` - Initial setup for development dependencies.
- `scripts/test` - Run test suite.
